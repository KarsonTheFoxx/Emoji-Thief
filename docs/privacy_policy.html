Privacy Policy

Welcome to the privacy policy, this document will cover how data is proccessed, used, and stored as well as give instructions on how to request data removal.
Before you read this document read the TOS first


Data collected

Emoji Thief will read the message content of the message that the command is ran on. This content includes only whatever the user can see from their Discord client.


Data Processing

Emoji Thief will process the contents of the message that the command is ran on to search for custom emojis. Once the scan is complete anything unrelated to a custom emoji is tossed out an not used further. When Emoji Thief resonds to the command all remaining information is thrown away.


Data Storage and Data Removal Requests

Emoji Thief does not store any information anywhere internally, all information used is publicly avialable to all users if you know where to look.
Because no data is stored there is currently no process to request data removal.
The author cannot prevent your emoji from being stolen with this app, it uses the Discord provided CDN servers; These servers serve the user embeded content such as images, gifs, and the icon to display for custom emojis. The only way to prevent an emoji from being stolen by this bot is to not use the emoji in servers this bot exists in, or remove members abusing this bot from your server.
NOTE: DISABILING USER APPS DOES NOT PREVENT THIS BOT FROM WORKING, THIS SETTING SIMPLY MAKES THE MESSAGE RENDER EMPHERIAL (ONLY VISIBLE TO THE USER WHO RAN THE COMMAND).
The author is not resposible for protecting your emojis from this bot.
